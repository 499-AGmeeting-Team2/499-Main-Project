# 499-Main-Project

## User group
The user group for this project is individuals who have the need of having efficient large-scale meetings with voting needs. It is meant to be used with other live video meeting applications.

## Requirements
- NodeJs
- React
- PSQL
- Redux
- Axios
- Webpack
- PassportJs
